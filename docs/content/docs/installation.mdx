## Requirements

- Node.js 20+
- TypeScript 5+
- NestJS 10+ or 11+

## Install the Package

<Tabs groupId="package-manager" items={['npm', 'pnpm', 'yarn']} persist>
  <Tab value="npm">

```bash
npm install nestjs-openapi --save-dev
```

  </Tab>
  <Tab value="pnpm">

```bash
pnpm add -D nestjs-openapi
```

  </Tab>
  <Tab value="yarn">

```bash
yarn add -D nestjs-openapi
```

  </Tab>
</Tabs>

## Peer Dependencies

The library requires `@nestjs/common`. If you use `@nestjs/swagger` decorators in your code, you already have it installed.

```json
{
  "@nestjs/common": "^10.0.0 || ^11.0.0"
}
```
